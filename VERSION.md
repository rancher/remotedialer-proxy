Rancher's remotedialer-proxy follows a pre-release (v0.x) strategy of semver. There no compatibility between releases. Instead, each remotedialer-proxy version aims to be compatible with a single version of Rancher. The current supported versions of the remotedialer-proxy (and the compatible version of rancher) are as follows:

| remotedialer-proxy Branch | remotedialer-proxy Minor version | Rancher Minor Version |
|---------------------------|-----------------------------------|----------------------|
| main | v0.9 | v2.16 |
| release/v0.8 | v0.8 | v2.15 |
| release/v0.7 | v0.7 | v2.14 |
| release/v0.6 | v0.6 | v2.13 |
| release/v0.5 | v0.5 | v2.12 |
| release/v0.4 | v0.4 | v2.11 |
